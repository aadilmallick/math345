Welcome

This repository contains simple static visualizations of SIR epidemic models. Available visualizations:

  • Interactive SIR Chart - Classic SIR model with charts
  • Interactive SIR Grid - SIR model with population grid visualization
  • SIR Grid with Vaccinations - SIRV model showing vaccination effects
  • SEIRV Grid with Vaccinations - SEIRV model with exposed period and vaccination effects

Notes for GitHub Pages

  • Pages are static — keep links relative (already done).
  • Enable GitHub Pages in your repository settings and serve from the main branch (root) or a gh-pages branch.
  • After enabling, your site will be available at https://<user>.github.io/<repo>/. The interactive chart will be at /sir-charts.html.
